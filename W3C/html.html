# html 规范

# 代码规范

  1. 单标签不要写闭合标签，例如img,br,input
  2. ul/ol直接元素只能是li，li只能写在ul/ol里面
  3. 行内元素里面不可使用块级元素，a标签除外
  4. 文字类块级标签里面不能放其他块级元素
  5. 每个页面都要写<!DOCType html>，html要加上lang属性
  6. html要保持简介，标签嵌套的层数能少则少
  7. 使用合适的标签
  8. 如果是按钮就应该写一个button或者 input type=”button”，而不是写一个a标签设置样式
  9. 如果是下划线就应该使用border，而不是写一个固定高度的占位标签
